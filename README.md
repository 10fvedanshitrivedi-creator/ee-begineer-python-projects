# ee-begineer-python-projects
# Electrical Engineering Python Projects

This repository contains beginner-level Electrical Engineering projects built using Python.

## Projects

### 1. Electricity Bill Calculator
- Calculates electricity bill from meter readings
- Includes slab system
- Tax and subsidy support

### 2. Ohm’s Law Calculator
- Calculates Voltage, Current, or Resistance based on input
- Multi-mode calculator system
- Input validation

### 3. Power Calculator
- Calculates Power using different combinations of V, I, and P
- Menu-driven program
- Basic electrical formula implementation

## Tech Used
- Python
- Basic Electrical Engineering concepts

## Purpose
To combine Electrical Engineering fundamentals with programming for practical understanding.
